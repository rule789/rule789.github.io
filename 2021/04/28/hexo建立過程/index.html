    hexo建立過程

    發表於 2021-04-28

    安裝

    1. 要先安裝 Node.js、Git

    2. 安裝 hexo

    3. 建立 hexo 專案

    4. 可以去改配置 in _config.yml

    5. 輸入以下指令建立網頁並開啟伺服器

    安裝主題

    範例：NexT

    npm install hexo-theme-next x

    建議用 git clone

    git clone https://github.com/next-theme/hexo-theme-next themes/next

    如果用 npm install 主題資料夾會在 node_modules 中，如果用 git clone，主題資料夾會在 themes 中。

    將 _config.yml 中的 theme 改成 next

    將主題的資料夾中的 _config.yml 裡面的 scheme 改成自己想要的

    (可詳主題的 github 的 readme)
